Unity Pathfinding Entity

I consider this a big project
because I was really proud of how the project came together
this project was for SHINE an expo where we show how we grew from our past projects.
for my past project I tried to make a some-what scary game which I will talk about later on this page.
On the image below this text you can see a maze where the player would walk through whilst a red entity chases the player
(E being the entity, the blue line being the path the entity takes and P being the player)
the closer the entity got to the player the louder scary music would play.
The entity's pathfinding was made to get the optimal path towards the player and it worked too.

Ambition Project

This is the project I made BEFORE making the Unity pathfinding project,
this is my ambition project where I wanted to make something eerie so it sticks with the player for longer.
At the time that I made this I found unity still rather difficult to use so I opted to use something more familiar first, that being LUA.
LUA is a pretty popular coding language mostly known for being used in games on the ROBLOX platform.
Even though I opted out of something more difficult which I shouldn't have I still really liked the outcome of this project.
The main reason why I liked the outcome is that I could see what you could do with creativity and a little bit of code.
The game that I made for this project is up for the public to play and 5 people have it favorited which also makes me glad,
if you wish to play it too I have put the link to the game right here.
